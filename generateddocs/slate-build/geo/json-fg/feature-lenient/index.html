    JSON-FG Feature - Lenient ogc.geo.json-fg.feature-lenient

    A OGC Features and Geometries JSON (JSON-FG) Feature that does not require the "time" and "place" properties.

    Status: Stable

    This building block is valid

    Description

    OGC Features and Geometries JSON (JSON-FG) extends GeoJSON to support a limited set of additional capabilities that are out-of-scope for GeoJSON, but that are essential or important for a variety of use cases involving feature data.

    This Building Block extends the standard JSON-FG one by removing the requirement to provide values for the "time" and "place" properties.
